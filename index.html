The matchup chart looks like this:
Run from -> Team -> Attack
Pink, Blue, Orange -> Red -> Yellow, Green, Purple
Blue, Orange, Red -> Yellow -> Green, Purple, Pink
Orange, Red, Yellow -> Green -> Purple, Pink, Blue
Red, Yellow, Green -> Purple -> Pink, Blue, Orange
Yellow, Green, Purple -> Pink -> Blue, Orange, Red
Green, Purple, Pink -> Blue -> Orange, Red, Yellow
Purple, Pink, Blue -> Orange -> Red, Yellow, Green
Red, Orange, Yellow, Green, Blue, Purple, Pink -> White ->
White is a shy team, will only run away from everybody, and has an extended vision radius

Red runs from Blue, Orange, Pink
Red attacks Yellow, Green, Purple

White will start with ten circles
Every other color will start with six circles
If a color is tagged by an attacking circle (i.e. Red tags Green) the tagged circle will convert to the attackers team (Green joins Red team)
The decision of which circle joins which team when collision occurs is decided by the matchup chart
The larger canvas allows for multiple "chases" to happen at once
